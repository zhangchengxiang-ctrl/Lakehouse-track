# GeoIP 数据库

将 `GeoLite2-City.mmdb` 放入此目录以启用 Vector 的 IP 地理位置解析。

**下载方式**：
- 国内镜像: `wget https://ghproxy.com/https://github.com/P3TERX/GeoLite.mmdb/raw/download/GeoLite2-City.mmdb -O GeoLite2-City.mmdb`
- 或直接: https://github.com/P3TERX/GeoLite.mmdb/raw/download/GeoLite2-City.mmdb
- 官方: https://www.maxmind.com （需注册获取）
